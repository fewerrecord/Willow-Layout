Privacy Policy

Willowcreek Cafe uses this application internally to coordinate table assignments among its own servers and managers. The only personal information collected for SMS purposes is a server's name and mobile phone number, used solely to send an automated text alert when that server has been assigned a table during their shift.

A manager enrolls a server's phone number from within the app. Before any alert texts are sent, the server must reply YES to a one-time confirmation message to opt in. A server can reply STOP at any time to immediately stop receiving texts, or HELP for support, and a manager can also disable SMS notifications for any server within the app at any time.

Message frequency varies with shift activity. Message and data rates may apply. Mobile phone numbers and any other information collected through this SMS program are never sold, shared, or disclosed to third parties, and are never used for marketing of any kind.
