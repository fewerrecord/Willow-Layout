Privacy Policy

This is a personal hobby project I built to coordinate table assignments among a small group of users I know personally. The only personal information collected is a user's name and mobile phone number, used solely to send a text when that user has been assigned a table. The app is shared among this small group, and any user can add, edit, or remove any server's details, including phone numbers, at any time. SMS notifications can be toggled on or off within the app. This information is never sold, shared, or used for marketing of any kind.
